Hash Util

Work with file hashes easier than ever


Super easy install

curl -fsSL https://raw.githubusercontent.com/Proactive-Development/hash-util/main/scripts/install.sh | bash -

See all the hashes of your files

user@computer:~# hash-util -ls
File: README.md MD5:e2cf67ded752eaeec784e62fbf76d5c6
SHA224:7782b31e5c33ca07fde01662c63657547806faaecbadf2196035dbf2
SHA256:7ec39b57403ff41f418b00cbfc117067e5d54c6ba389bb210c993099cc9c958f
SHA384:fe393e9e66e9eb177172170aa957751136e152212a6cee64cb2c5e8c9ec032c8cad575cdb294b39ec55597db2b6e1386

Find files with hashes

user@computer:~# hash-util -f 6c4e0639985cc7dcc7c11d385b341e3fs
['/hash-util/README.md']

Verify files with ease

user@computer:~# hash-util -v README.md 6c4e0639985cc7dcc7c11d385b341e3f
[PASS] File has the same hash as the given hash: 6c4e0639985cc7dcc7c11d385b341e3f

CaesarPlus

An advanced caesar cypher python module


Quick install with pip

pip install caesarplus

Encrypt any text

caesarplus.encode("Example")
([4, 4, 1, 7, 5, 0, 6],'I|btulk')

Decrypt any text with the key and encrypted text

caesarplus.decode([4, 4, 1, 7, 5, 0, 6],'I|btulk')
"Example"
